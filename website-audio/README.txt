# เว็บไซต์เล่นเสียง

เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ (Chrome/Edge/Firefox/Safari) แล้วกดปุ่ม ▶ เล่น หรือใช้ตัวควบคุมของเครื่องเล่นเสียงได้เลย

## โครงสร้างไฟล์
- index.html — หน้าเว็บหลัก
- audio/voice.mp3 — ไฟล์เสียงของคุณ

> หมายเหตุ: หากต้องการเปลี่ยนไฟล์เสียง ให้แทนที่ `audio/voice.mp3` เป็นไฟล์ใหม่ชื่อเดียวกัน
